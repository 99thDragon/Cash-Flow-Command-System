# Cash Flow Command System

A professional cash flow management system for small businesses and solo founders. Track accounts receivable, accounts payable, and make informed cash flow decisions with beautiful visualizations and automated workflows.

## Features

- **Real-time Cash Position Tracking** - Monitor your current cash balance and runway
- **12-Week Cash Flow Forecast** - Extended visibility for better planning
- **Accounts Receivable Management** - Track invoices with aging and automated reminders
- **Accounts Payable Tracking** - Manage bills with priority ranking
- **Recurring Transaction Templates** - Automate monthly rent, retainers, and subscriptions
- **CSV Import** - Bulk import bank transactions
- **Decision Simulation Tool** - "Can we afford X?" scenario planning
- **Weekly Review Wizard** - Guided routine for cash flow accountability
- **Dark Mode** - Beautiful UI with theme switching
- **Data Visualization** - Interactive charts powered by Chart.js

## Quick Start

1. Open `index.html` in your browser
2. Start adding invoices and bills
3. Set up recurring templates for predictable expenses
4. Import historical data via CSV
5. Review your 12-week forecast
